## 스프링 팩키지 다운로드       
https://start.spring.io/        
     
- Project > Gradle        
- Language > Jva        
- spring Boot > 3.4.5          
- group > 프로젝트명 입력     
- Packaging > jar 선택     
- dependencies 아래 4개 추가         
  Spring Web        
  H2 Database   
  Mustache   
  Sprign Data JPA   
     
- Generate  클릭후 다운로드 압축풀기
